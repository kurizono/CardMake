[トランプのサイズ]

標準的なトランプのカードの大きさには、ブリッジサイズとポーカーサイズの二種類がある。
ブリッジサイズ
約57mm×約89mm (2 1⁄4 × 3 1⁄2 in)。横幅が狭いので、手に持つ枚数が多いゲームに適している。
ポーカーサイズ
約63mm×約88mm (2 1⁄2 × 3 1⁄2 in)。横幅が広いので、手に持つ枚数が少ないゲームに適している。ISO 216のB8とほぼ同じである。
